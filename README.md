## Usage
Please look https://docs.python.org/ko/3/library/venv.html
```
# For bash
source ./.venv/bin/activate

# For windows (Powershell)
PS C:\> <venv>\Scripts\Activate.ps1

# Install requirements if not installed
pip install -r requirements.txt
```
